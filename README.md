# ReFlex
It is a new Modelling Creation Tool for RedoxOS

+ VULKAN API, MetalAPI
+ Completly written in Rust!
+ Parallel Coding, Multithreading
+ Without OpenGL or DirectX Support

## Support

+ LINUX (Wayland Server)
+ MAC OS
+ WINDOWS
+ ANDROID x64
+ REDOX OS

## Features

+ 3d printing
+ Node based Texturing
+ Global Illumination
+ OpenCollada Support
